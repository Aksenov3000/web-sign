Про настройки клиента

Для работы с Рутокен ЭЦП 2.0 и 3.0, на клиентской машине необходим только rutoken-plugin (https://www.rutoken.ru/support/download/rutoken-plugin/)
Никакого дополнительного софта не требуется  (драйвера Рутокен, КриптоПро CSP, КриптоПро ЭЦП browser plugin). 
Единственное, что сам плагин ставит расширение в браузеры по умолчанию, и если оно не встало, его надо доставить руками.https://chrome.google.com/webstore/detail/%D0%B0%D0%B4%D0%B0%D0%BF%D1%82%D0%B5%D1%80-%D1%80%D1%83%D1%82%D0%BE%D0%BA%D0%B5%D0%BD-%D0%BF%D0%BB%D0%B0%D0%B3%D0%B8%D0%BD/ohedcglhbbfdgaogjhcclacoccbagkjg?hl=ru
Для других браузеров видимо нужно будет устанавливать другие расширения...
Но если дополнительный софт стоит - то тогда сертификаты могут дублироваться, т.к. получается информация об одном и томже из разных источников

Для работы Рутокен lite нужно - ...

1. Для КриптоПро
КриптоПровайдер - КриптоПро CSP

Настройка проекта

yarn        - Это устанавливает все зависимости
yarn dev    - Запуск отладочного веб-сервера
yarn build  - Компиляция статики для прода. Создаются две папки:
                - build - это исходники билда, 
                - dist - это итоговый результат склейки
